**Welcome to spectrum-split-FrontEnd project** 

**About**


This project contains everything you need to run your app locally.

**Edit the code in your local development environment**


**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
The project is the frontend of a six-stem audio splitter. The backend is available at https://github.com/magnusAchor/spectrum-split-backend
. Clone the Python backend repository using git clone. It uses Demucs to split audio files.
```

Run the app: `npm run dev`
